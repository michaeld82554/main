# ConfidentialPay

ConfidentialPay is a privacy-first payroll and treasury platform concept designed for companies that need secure, efficient, and compliant financial operations across distributed teams.

The product aims to simplify payroll, treasury oversight, and employee management while supporting a stronger foundation for private and cross-chain payment workflows.

## Project structure
- src/ - frontend application
- backend/ - API and server logic
- public/ - static assets

## Getting started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the app:
   ```bash
   npm run dev
   ```
   This runs the frontend on port 3000 and the backend in watch mode.
